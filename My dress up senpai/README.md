# Dress Up Game ✅

Simple browser-based dress-up prototype.

How to run
- Open `index.html` in your browser (double-click or use `open index.html` on macOS).

Controls
- Choose a **Model** at the top of the sidebar to switch the base model (switching resets the outfit).
- Use the **Wardrobe** categories (All/Head/Top/Bottom/Shoes) to filter available items.
- Click an item in the left panel to add it to the model.
- Click an added item to select it; drag to move; use sliders in the inspector to scale and rotate.
- Use **Reset** to clear, **Randomize** to create a random outfit, and **Save Image** to download a PNG.

Keyboard shortcuts
- **R**: Reset the outfit
- **Z**: Randomize
- **S**: Save image
- **Delete**: Remove selected item

Notes
- All assets are simple SVGs included in `assets/` and are free to edit.
- This is a minimal prototype—happy to add layering handles, snapping, or a wardrobe system next.
